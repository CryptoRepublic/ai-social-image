# ✨ AI Image Generator

A simple, lightweight web application to generate AI images for free, powered by the fast and reliable [Pollinations.ai](https://pollinations.ai/) API.

## 🚀 Features
- **Minimalist & Modern UI**: Clean design focused on user experience.
- **No Sign-Up / No API Keys**: Completely free to use.
- **Multiple Formats**: Choose between Square, Landscape, and Portrait.
- **Regenerate Button**: Instantly create a new variation of your prompt.
- **Download Button**: Save the generated image directly to your device.
- **No Watermarks**: Images are generated cleanly (`nologo=true`).

## 🛠️ Technologies Used
- **HTML5** for structure.
- **CSS3** for styling.
- **Vanilla JavaScript** for logic and direct file downloading.
- **Pollinations API** for image generation (No backend needed).

## 💻 How to run locally
1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.
3. Type your prompt, select your preferred format, and click "Generate Image".

## 🌐 Deploy to Netlify
This app is entirely static and can be hosted for free on Netlify in just a few clicks:

1. Push these files to a public or private GitHub repository.
2. Log in to [Netlify](https://www.netlify.com/).
3. Click on **"Add new site"** -> **"Import an existing project"**.
4. Choose **GitHub** and authorize access to your repositories.
5. Select your repository.
6. Leave the "Build command" and "Publish directory" fields empty (since `index.html` is in the root).
7. Click **"Deploy site"**.

Your app will be live on a custom Netlify URL, ready to be shared with the world!
